# EmbedAnything
Mediawiki extension to embed media into a page

* Use with:

```
<embed>URL</embed>
```
OR
```
<embed url=URL></embed>
```
OR
```
{{#tag:embed|URL}}
```

* Use "template" or "style" attribute to select template to use for embedding. ie:

```
<embed url=URL template=TEMPLATE></embed>
```
